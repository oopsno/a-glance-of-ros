# ROS通信机制

ROS的通信机制主要实现在Github仓库[`ros/ros_comm`][repo_ros_comm]中。该仓库主要包括

+ [`rospy`][src_rospy]
+ [`roscpp`][src_roscpp]
+ [`roslisp`][repo_roslisp]

等三个客户端模块和

+ [`genpy`][repo_genpy]
+ [`gencpp`][repo_gencpp]
